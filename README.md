# 202-Happy-Number-leetcode
A number is called happy if we repeatedly replace it with the sum of the squares of its digits and eventually reach 1; if the process enters a loop without reaching 1, the number is not happy. EX : input n=19 12 + 92 = 82 82 + 22 = 68 62 + 82 = 100 12 + 02 + 02 = 1 then output : true  ; Input: n = 2 Output: false
